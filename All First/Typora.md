## yml 配置  支持PicGo

路径 `C:\Users\H433899\AppData\Roaming\Typora\typora.yml` ，默认调用PicGo 的server 模式

```
image:
  upload: true
  service: picgo-server
  picgo-server:
    url: "http://127.0.0.1:36677/upload"
```
